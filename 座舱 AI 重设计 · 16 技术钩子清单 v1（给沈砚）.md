**链路节点定义**

|  |  |  |
| --- | --- | --- |
| 节点 | 名称 | 涵盖能力 |
| ① | **输入层** | 物理信号采集：麦克风、摄像头、传感器、触摸屏、CAN 信号 |
| ② | **感知层** | 信号处理与结构化：ASR、唤醒、音区分离、DMS/OMS、手势、声音事件 |
| ③ | **理解层** | 意图与上下文：NLU、多轮对话、指代消歧、情境推理 |
| ④ | **决策层** | 任务规划：主动服务、Agent 编排、MCP 路由、复杂指令分解 |
| ⑤ | **执行层** | 工具调用：车控、导航、音乐、空调、第三方 MCP、跨端 |
| ⑥ | **输出层** | 反馈呈现：TTS、HMI 多屏、虚拟形象、触觉、灯光 |
| ⑦ | **基础设施层** | 支撑能力：大模型车云协同、个性化、弱网降级、安全合规、数据闭环 |
