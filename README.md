## 设计哲学

**Harness Engineering**：给 AI 引擎装线束。七大线束：状态机 / 置信度阈值 / 降级路径 / 安全红线 / 可观测性 / 人类监督 / 失败隔离。

**核心原则**：安全优先 → 端云协同 → 失败优雅 → 可观测 → 简单。
