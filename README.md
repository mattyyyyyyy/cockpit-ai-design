## 项目概述

面向智能座舱的车载 AI 助手系统，覆盖语音交互、意图识别、多工具编排、主动服务、记忆系统等完整链路。核心能力包括：

- **语音交互**：ASR（FunASR SenseVoice）→ NLU → Agent → TTS，支持全双工打断
- **工具生态**：星巴克点单、肯德基点餐、淘宝购物、电影票、导航、充电、音乐等 MCP 工具
- **记忆系统**：四层记忆（工作记忆 / 用户偏好 / 任务状态 / 节律学习）
- **Agent 架构**：CLAW 闭环框架，Mother-bot + 分域 Sub-bot，支持模糊指令、多指令、纠错、补充
- **安全保障**：Harness 线束层，操作风险分级 + 10s 撤销窗口 + 熔断器 + 幂等保护
